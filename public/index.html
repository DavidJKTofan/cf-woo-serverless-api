Serverless WOO API

A Cloudflare Workers-powered RESTful API for curated resources.

For educational purposes only.

Parameters

/api/resources    Request JSON data from Cloudflare's edge servers.
Path params:
/                 Returns all.
#                 ID Number. Valid for integer between 0 and 381.
?category=X       Filter by category (e.g., WEB%20DEV, CYBERSEC).

Example

$ curl https://api.cf-testing.com/api/resources/32
{
  "id":32,
  "title":"Bitbucket",
  "description":"Bitbucket is more than just Git code management",
  "url":"https://bitbucket.org/product",
  "main_cat1":"WEB DEV",
  "main_cat2":"",
  "tag1":"DEVELOPMENT",
  "tag2":"PLATFORM",
  "tag3":"GIT CODE"
}
